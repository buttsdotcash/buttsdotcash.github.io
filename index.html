"What is the point of this website?"

I'm so glad you asked! It is highly likely you're reading this because I told you I owned the URL butts.cash (as is the natural impulse that occurs in a person who owns such a domain name) and, either disbelieving or intrigued, you loaded up this website to see what it contained. Prior to the existence of this website, when this interaction happened, anyone attempting to load this website would be met with a parked page. This seemed like a poor use of such a good domain name, so I decided I had to put at least something on here. This is that something!



"Is it just pictures of this guy?"

When I was planning this, I decided it was important to have pictures of myself on the website. If the whole point of this website is proving to people that I own the URL butts.cash, there has to be some kind of evidence. So, in some senses, yes, this website is just pictures of me. Although, if you click around, you might find something else.
